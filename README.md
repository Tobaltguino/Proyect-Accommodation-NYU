# Sistema de Alojamiento Universitario NYU 🏢

Proyecto semestral desarrollado para el ramo de **Taller de Aplicaciones Web** (Universidad de Tarapacá). Este sistema permite gestionar las solicitudes, asignaciones, estadías e incidencias de los residentes universitarios, integrando validaciones de estado, pagos y planes alimenticios con otros equipos.

## 🛠️ Stack Tecnológico

* **Frontend:** Angular
* **Backend:** NestJS (TypeScript)
* **Base de Datos:** PostgreSQL alojada en Supabase
* **ORM:** TypeORM (Enfoque Code-First)

## ⚙️ Requisitos Previos

Asegúrate de tener instalado en tu entorno local:
* [Node.js](https://nodejs.org/) (v18 o superior)
* npm (gestor de paquetes)
* Angular CLI (`npm install -g @angular/cli`)
* NestJS CLI (`npm install -g @nestjs/cli`)
